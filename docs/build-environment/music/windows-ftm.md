# music Windows(FamiTracker) 환경

## 대상 파일

- `music/DOOM.ftm` (편집용 프로젝트)
- `music/DOOM.nsf` (재생/배포용 산출물)

## 준비

- FamiTracker 또는 호환 에디터 설치

## 작업 절차

1. `DOOM.ftm` 열기
2. 곡/패턴 편집
3. NSF Export로 `DOOM.nsf` 생성
4. 필요 시 `nesrom` 음악 드라이버와 포맷 호환성 확인

## 검증

- NSF 플레이어(Foobar2000 + NSF 플러그인 등)에서 재생 테스트
- 실제 NES ROM 통합 시 타이밍/트랙 전환 확인
