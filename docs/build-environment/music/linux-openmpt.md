# music Linux 환경 (대체 도구 중심)

`DOOM.ftm`은 전통적으로 Windows FamiTracker 생태계가 가장 안정적입니다.
Linux에서는 아래 우회 전략을 권장합니다.

## 전략 1: Wine으로 FamiTracker 실행

- Wine 설치 후 FamiTracker 실행
- `.ftm` 편집 및 `.nsf` export 수행

## 전략 2: Windows VM/원격 환경 병행

- Linux에서 버전 관리, Windows에서 실제 export

## 전략 3: NSF 결과물만 관리

- Linux에서는 `DOOM.nsf` 재생 검증 위주로 운영
- 편집은 Windows 전담

## 결론

- 음악 편집/내보내기는 Windows 중심, Linux는 검증/관리 중심으로 역할 분리하는 것이 현실적입니다.
